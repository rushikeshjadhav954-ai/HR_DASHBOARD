# 📊 HR Analytics Dashboard

## 📌 Project Overview
Analyzed HR dataset containing 22,000+ employee records (2000–2020) to uncover workforce trends, employee demographics, and attrition patterns. Data cleaning was performed using MySQL, and an interactive dashboard was built using Power BI.

---

## 🛠️ Tools Used
- MySQL Workbench
- Power BI

---

## 📊 Dataset
- HR dataset with 22,000+ rows
- Time period: 2000–2020

---

## ❓ Business Questions
- What is the gender breakdown of employees?
- What is the race/ethnicity distribution?
- What is the age distribution?
- HQ vs Remote employee count?
- Average tenure of terminated employees?
- Which department has highest turnover?
- State-wise employee distribution?
- Employee growth over time?

---

## 📈 Key Insights
- Majority employees are male
- Most employees belong to White ethnicity
- Largest age group: 25–34
- Headquarters has more employees than remote
- Average tenure of terminated employees ~7 years
- Marketing department has highest turnover
- Most employees are from Ohio
- Employee count increased over time

---

## ⚠️ Data Cleaning
- Excluded 1599 future termination dates
- Standardized date formats using SQL
- Handled NULL and missing values
